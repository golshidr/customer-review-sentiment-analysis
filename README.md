# Sentiment Analysis for Customer Reviews

This project uses machine learning to analyze customer reviews and classify them as positive or negative.

The goal is to demonstrate how natural language processing (NLP) can be used to extract meaningful insights from large volumes of customer feedback.

---

## Dataset

The dataset used in this project is the IMDb movie review dataset containing 50,000 labeled reviews.

Each review is labeled as either positive or negative.

---

## Methodology

The project follows these steps:

1. Text preprocessing and cleaning
2. Converting text to numerical features using TF-IDF
3. Training a Logistic Regression classification model
4. Evaluating model performance
5. Extracting important words influencing sentiment

---

## Results

The sentiment classification model achieves high accuracy in distinguishing positive and negative reviews.

The analysis also identifies words that strongly influence sentiment predictions.
